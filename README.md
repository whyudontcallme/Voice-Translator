# Голосовой Переводчик

Веб-приложение для записи речи и её перевода на несколько языков одновременно.

## Особенности

- 🎤 Голосовое распознавание речи (SpeechRecognition API)
- 🌍 Поддержка 16 языков
- 📝 Одновременный перевод на несколько целевых языков
- 🔊 Воспроизведение переводов голосом
- 🎨 Тёмная минималистичная дизайн
- 🌐 Автоматическая локализация интерфейса в зависимости от выбранного языка ввода

## Поддерживаемые языки

1. Русский (ru)
2. Английский (en)
3. Испанский (es)
4. Французский (fr)
5. Немецкий (de)
6. Итальянский (it)
7. Португальский (pt)
8. Японский (ja)
9. Китайский (zh)
10. Корейский (ko)
11. Арабский (ar)
12. Хинди (hi)
13. Украинский (uk)
14. Польский (pl)
15. Турецкий (tr)
16. Шведский (sv)

## Требования

- Современный браузер с поддержкой:
  - Web Speech API (SpeechRecognition)
  - Web Speech API (SpeechSynthesis)
  - Fetch API
  - Интернет-соединение

## Использование

1. Откройте `index.html` в браузере
2. Выберите язык ввода (язык, на котором вы будете говорить)
3. Выберите целевые языки (на какие языки переводить)
4. Нажмите кнопку "Начать"
5. Говорите в микрофон
6. Результаты появятся автоматически

## Особенности языков

**Арабский и Хинди:** Поддержка SpeechRecognition в браузере может быть ограничена. Если у вас не работает распознавание речи для этих языков, это может быть ограничением браузера, а не приложения.

## Архитектура перевода

Приложение использует несколько API переводчиков в качестве fallback:
1. Google Translate (googleapis.com/translate_a)
2. MyMemory Translation API
3. Bing Translator (если доступно)
